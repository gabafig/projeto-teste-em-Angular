Olá!

Este é um projeto com objetivo de prática didática e formação de portfólio.


A proposta deste projeto é a prática da implementação das ferramentas e funcionalidades listadas abaixo.

Implementado em Novembro/2023
1 - Inserção de código HTML5 manual para construção desta página Home.
2 - Inserção de estilização manual (simples) via CSS3 desta página Home.
3 - Instalação e utilização do framework Angular 2 para criação de um Single Page Application, aprendendo sobre rotas, importação de dependências e sua arquitetura.
4 - Utilização do framework de layout Angular Material para utilização de templates e componentes pré-definidos na construção do projeto.
5 - Criação de um formulário de cadastro e de login, com validadores de campo.
6 - Utilização dos recursos de "Guard" para proteger rotas e então simular uma tela somente acessada por Login.
7 - Criação de um servidor fake via json-server para simular uma API que armazena as informações cadastradas e então criar um Service que irá acessar e validar essas informações.
8 - Validar as informações de Login no arquivo json e então, liberar a rota.
9 - Praticar comandos e rotinas Git e armazenamento no GitHub.


Este projeto deverá passar por melhorias constantemente e se tornar algo mais complexo com o tempo! :)
